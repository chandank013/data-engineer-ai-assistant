## 1️⃣ Clone the Repository

Open a terminal and run:


git clone https://github.com/chandan013/data-engineer-ai-assistant.git

### Move into the project directory
cd data-engineer-ai-assistant
---

## 2️⃣ Install Project Requirements

###Install the required Python dependencies:

pip install -r requirements.txt
---

## 3️⃣ Make Changes to the Project

### Open the project in your code editor and modify or add files as needed.

Example editors:

VS Code
---

## 4️⃣ Add Changes to Git

### Stage the changes:

git add .
---

## 5️⃣ Commit the Changes

### Write a meaningful commit message describing what you changed.

git commit -m "Describe the changes you made"
---

## 6️⃣ Push Changes to GitHub

### Since you have Write permission, you can push directly to the main branch.

git push origin main
---
